По выполнению лабораторной работы отчет предоставляется в виде:
- документ формата .doc (.docx), содержащий промежуточные копии экрана с комментариями;
- файл проекта GeoSCADA (.sde), полученный в ходе выполнения ЛР.

Задания для самостоятельного выполнения:

1. Создать систему управления уровнем выгребной ямы. В яму с постоянной скоростью поступает грязная вода. При достижении максимального уровня должен включаться насос для откачки воды. Он качает до какого-то минимального уровня и отключается. Кнопки управления насосом и индикация.

2. Создать систему управления прессом. Пресс прессует детали, когда опи перед ним есть. Детали нет - загарается аварийная лампа. Должна иметь два режима работы - автоматический и ручной. Количество подаваемых деталей задается.

3. Создать систему управления температурой в теплице. При достижении максимально допустимой температуры открывается форточка, и остается открытой до достижения минимально допустимой температуры. Кнопки управления электродвигателем форточки и индикация.

4. Создать систему питания воздухом. Состоит из трубопроводов, компрессора, ресивера, и кнопок управления (включения/выключения компрессора). Кнопки с индикацией. При достижении максимального давления (устанавливается преподавателем) компрессор выключается. При этом давление должно падать. Затем, при минимальном - включается.

5. Создать систему поддержания температуры в помещении. При минимальном значении температуры должен включиться нагреватель. С пульта управления должна настраиваться скорость нагрева, дискретно. То есть три положения, медленно, средне, быстро. В комнате, соответственно, три нагревателя. Скорость нагрева должна отличаться при одном включенном нагревателе, и при трех. Показатель температуры - аналоговый регулятор. При выключенных нагревателях помещение остывает с одной скоростью.

6. Создать емкость с двумя насосами. Первый наполняет, второй опустошает емкость. Скорости должны быть разные, чтобы при одновременном включении возникала аварийная ситуация, либо перелива, либо опустошения емкости. Два датчика ВНУ и ННУ. Две кнопки включения/выключения насосов, кнопка автоматического режима. В автоматическом режиме насосы включаются и отключаются самостоятельно. Если режим авто отключен. При аварии должно все отключиться.

7. Создать систему перемешивания. По нажатию кнопки пуск включается миксер, перемешивание идет в течение определенного времени. Как только таймер срабатывает, миксер выключается. Чтобы заново запустить систему необходимо нажать кнопку сброс, затем Пуск.

8. Создать систему наполнения бассейна. Наполняется из двух емкостей, сначала из одной, затем, когда первая опустошится включается второй насос и наполнение происходит из второй емкости. Когда бассейн наполнен, все выключается, кнопка сброс возвращает в начальное состояние. Кнопка пуск запускает цикл. Три аналоговых регулятора уровня (бассейна, емкость1, емкость2).

9. Создать систему охлаждения. Жидкость циркулирует по двум кругам, малый и большой. Когда движется по малому - нагревается, движется по большому - охлаждается. Кнопка пуск запускает цикл. Регулятор температуры на экране. При достижении максимальной температуры открывается клапан (условно) жидкость идет по большому кругу и остывает. Охладившись до необходимого уровня, начинает бегать по малому кругу.

10. Создать систему питания воздухом. Состоит из трубопроводов, компрессора, ресивера, потребителя (пневмоцилидра) и кнопок управления (включения / выключения / компрессора / пневмоцилиндра). Условия работы оговаривайте с преподавателем. Кнопки с индикацией. При достижении максимального давления (устанавливается преподавателем) компрессор выключается. При работе цилиндра давление должно падать. Затем, при минимальном - включается.
